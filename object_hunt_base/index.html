This page offers information about all used classes, namespaces and header files for the base process. The base process controls all attached hardwaer directly. It receives requests and commands via a TCP connection and is able to send a response. In a usual use case the base process is expected to build a connection to the ../../object_hunt_navigation/html/index.html "navigation process".
