Detailed Description

Helps handling multiple socket connections.

Administrates socket connections dynamically. Registers new connections and removes closed connections.

◆ setNavigationControlPipe()

void ConnectionManager::setNavigationControlPipe ( int  fd)

Sets the navigation control pipe.

Sets the file descriptor to the writing end of the control pipe of the Navigator class. This file descriptor is needed to notify the Navigator class about new connections. The class will only start working, if a valid file descriptor is set. Otherwise no action can be expected.

Parameters
fdA valid file descriptor of the writing end of the control pipe.
